# Alphafold Drug Platform

Alphafold Drug Platform allows scientists and researchers to utilize the power of Alphafold for drug discovery.

## Features
- Protein structure predictions using Alphafold 2.
- Drug-like compound screening.
- Integration with various databases and tools.

## Getting Started
To get started, follow the instructions below:

1. Clone the repository:
   ```bash
   git clone https://github.com/danjjak-ai/alphafold-drug-platform.git
   cd alphafold-drug-platform
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```
